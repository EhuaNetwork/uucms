UU建站系统
===============

UUCMS是一个免费开源的，快速、简单的轻量级PHP建站系统，专为懒人设计。UUCMS从诞生以来一直秉承简洁实用的设计原则，在保持出色的性能和至简代码的同时，更注重易用性。
> UUCMS的运行环境要求PHP7.2以上,支持二开。 二开详细开发文档参考 [ThinkPHP5完全开发手册](http://www.kancloud.cn/manual/thinkphp5)
